
Landslide Prediction and Warning System using ESP32

 Overview
An IoT-based system that predicts landslides using soil and weather data and sends real-time alerts via Telegram.



 Features
- ESP32-based monitoring
- Soil moisture & stiffness sensing
- Weather data (ThingSpeak)
- Telegram alert system
- Real-time risk prediction



 Project Structure
- esp32_code/ → ESP32 code  
- web/ → Weather data upload  
- images/ → Project visuals  
- docs/ → Diagrams
